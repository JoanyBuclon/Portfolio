# Portfolio Astro

My personal portfolio accessible [here](https://joanybuclon.com).

## Built With

This project uses the following stack:

- [Astro](https://astro.build)
- [Tailwind](https://tailwindcss.com/)
- [WebGL](https://developer.mozilla.org/fr/docs/Web/API/WebGL_API)
- [Docker](https://www.docker.com)

## Usage

Clone the repository and run `pnpm i`

You can then launch it using `pnpm dev`

## Deployment

This app is currently deployed via github actions over at arktos-01.
